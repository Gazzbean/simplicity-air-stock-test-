Setup guide
1
Set manager password & van regos
Open Settings as manager. Set a secure password and enter all 4 van registration plates. Save.
2
Techs create their PIN on first use
Each technician opens the app, selects their role, picks their van, enters their name, then creates a 4-digit PIN. The PIN is stored securely on their device only.
3
Import spreadsheet to Google Sheets
Upload your Rev3 file. Tab names must match: Master Parts, Van 1 Stock…Van 4 Stock, Usage Log, Reorder Log.
4
Enable Google Sheets API & deploy Apps Script
Google Cloud → enable Sheets API → get API key. In the Sheet → Extensions → Apps Script → paste google-apps-script.js → Deploy as Web App → paste URL in Settings.
5
Add to iPhone home screen
Safari → Share (□↑) → Add to Home Screen. Android: Chrome → ⋮ → Add to Home Screen.
